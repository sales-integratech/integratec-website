
# Integra Tech Solutions - Static Website

This is a ready-to-deploy static website for Integra Tech Solutions.
Files:
- index.html
- style.css
- script.js
- images/logo.png

Notes:
- The site uses external Unsplash images for hero/about/services. You can replace them by placing files in images/ and updating the HTML.
- To deploy:
  1. Create a GitHub repo and push these files.
  2. Import the repo into Vercel and deploy (Vercel will serve the static site).
- Contact form is demo-only. Use Formspree, Netlify Forms, or a server endpoint for production.
